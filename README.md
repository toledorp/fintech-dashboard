# AIRBank - Fintech Dashboard

Projeto desenvolvido como desafio técnico para simular um painel financeiro de controle de entradas e saídas.

## Tecnologias

- React
- TypeScript
- Vite
- Axios
- TanStack Query
- JSON Server
- React Hook Form
- Zod

## Funcionalidades

- Listagem de transações
- Integração com API mock
- Estrutura inicial do CRUD
- Base para dashboard financeiro
